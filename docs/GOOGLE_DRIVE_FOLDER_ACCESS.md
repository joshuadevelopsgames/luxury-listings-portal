## Step-by-Step Setup Instructions

### 1. Open the Folder
Go to: https://drive.google.com/drive/folders/1ld1rW8eTnWth2UDfRBrqGywZzsEPOJJw

### 2. Click "Share" Button
- Located in the top-right corner of the folder

### 3. Set General Access
- Click the dropdown next to "General access"
- Select **"Restricted"** (only people with access can view)
- **IMPORTANT**: Make sure "Anyone with the link" is NOT enabled

### 4. Add People with Editor Access

Add these emails as **"Editor"**:

```

```

**For each person:**
1. Type their email in the "Add people and groups" field
2. Click the dropdown next to their name
3. Select **"Editor"** (can upload, view, edit, delete)
4. Uncheck "Notify people" if you don't want to send emails
5. Click "Share"

### 5. Add People with Viewer Access

Add these emails as **"Viewer"**:

## Adding New Team Members

### When a New Social Media Manager Joins:
1. Add their Google email as **"Editor"**
2. They'll be able to upload and manage contracts

### When a New Content Director Joins:
1. Add their Google email as **"Viewer"**
2. They'll be able to view contracts but not edit

### When Someone Leaves:
1. Go to folder → "Share" → Find their email
2. Click the dropdown → "Remove access"
3. This revokes their access immediately

## Security Best Practices

### ✅ **Do:**
- Keep "General access" set to **"Restricted"**
- Only add people who need access
- Use "Editor" for Social Media Managers and Admins
- Use "Viewer" for Content Directors
- Remove access when people leave

### ❌ **Don't:**
- Enable "Anyone with the link" access
- Share the folder publicly
- Give "Editor" access to people who only need to view
- Forget to remove access when people leave

## Troubleshooting

### "I can't upload contracts"
- Check that your email has "Editor" access
- Verify you're signed into the correct Google account
- Contact admin to verify permissions

### "I can't see the folder"
- Make sure you've been added to the folder
- Check that you're signed into the correct Google account
- Verify the folder link is correct

### "The folder is publicly accessible"
- Go to folder → "Share" → Set to "Restricted"
- Remove "Anyone with the link" if enabled
